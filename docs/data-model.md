# Модель Данных

Документ описывает основные сущности системы расчета загрязнения тяжелыми металлами.

## Основные Сущности

### Площадка Пробоотбора

Площадка пробоотбора хранится в таблице `locations`.

Поля:

- `id`: идентификатор площадки;
- `name`: название площадки;
- `site_number`: номер площадки;
- `distance_from_source_km`: расстояние от источника выбросов в километрах;
- `description`: описание;
- `latitude`: широта;
- `longitude`: долгота;
- `created_at`: дата создания записи.

Площадки используются:

- для отображения точек на карте;
- для просмотра проб;
- как опорные точки при расчете поля концентрации;
- в статистике по годам и металлам.

### Металл

Справочник металлов хранится в таблице `metals`.

Поля:

- `id`: идентификатор металла;
- `name`: название металла;
- `symbol`: химическое обозначение;
- `unit`: единица измерения.

Справочник используется в пробах, фильтрах, статистике и расчетах.

### Проба

Данные концентрации металла хранятся в таблице `samples`.

Поля:

- `id`: идентификатор пробы;
- `metal_id`: ссылка на металл;
- `location_id`: ссылка на площадку пробоотбора;
- `position`: вид или положение пробы;
- `repetition`: повторность;
- `value`: значение концентрации;
- `sampling_date`: дата отбора;
- `analytics_number`: номер аналитики;
- `created_at`: дата создания записи.

На основе проб строятся:

- таблицы концентраций по выбранной площадке;
- фильтры по году и металлу;
- статистические графики;
- исходные значения для расчета поля загрязнения.

### Расчет

Расчет поля концентрации хранится в таблице `concentration_jobs`.

Поля:

- `id`: UUID расчета;
- `reference_location_id`: опорная площадка;
- `metal_id`: металл;
- `sample_year`: расчетный год;
- `grid_step_km`: шаг расчетной сетки;
- `area_size_km`: размер области расчета;
- `status`: статус расчета;
- `error_message`: текст ошибки;
- `created_at`: дата постановки расчета;
- `completed_at`: дата завершения.

Статусы:

- `queued`: расчет ожидает выполнения;
- `processing`: расчет выполняется;
- `completed`: расчет завершен;
- `failed`: расчет завершился ошибкой;
- `publish_failed`: расчет не был передан в очередь обработки.

### Точки Расчетной Сетки

Результат расчета хранится в таблице `concentration_grid_points`.

Поля:

- `job_id`: ссылка на расчет;
- `latitude`: широта точки сетки;
- `longitude`: долгота точки сетки;
- `concentration`: рассчитанная концентрация.

Эти данные используются для визуализации поля загрязнения на карте и для отчетов.

## Связи Таблиц

```text
locations 1 --- N samples
metals    1 --- N samples

locations 1 --- N concentration_jobs
metals    1 --- N concentration_jobs

concentration_jobs 1 --- N concentration_grid_points
```

## Индексы

В схеме предусмотрены индексы:

- `idx_samples_location`: быстрый поиск проб по площадке;
- `idx_samples_metal_year`: фильтрация по металлу и дате отбора;
- `idx_grid_points_job`: чтение точек результата по расчету.

## Принцип Хранения Результатов

Результаты расчета сохраняются отдельно от исходных проб. Это позволяет:

- повторно открывать уже выполненные расчеты;
- строить карту без повторного вычисления;
- экспортировать результаты в Excel;
- хранить историю расчетов за разные годы и металлы.
